# ml-podcast

A bilingual Machine Learning learning site built with Next.js 16.

## Stack

- Next.js 16 App Router
- TypeScript
- Local MDX content with a repository abstraction for future CMS migration

## Content model

- `content/series/*.json` for learning paths
- `content/episodes/*.mdx` for podcast/article style lessons
- `content/resources.json` for curated tools, papers, and datasets

## Commands

```bash
npm install
npm run dev
npm run build
```

Node.js is not available in the current shell environment, so dependencies were not installed or validated here.
