## Algorithm

`KNeighborsClassifier`

## Run This Code

```python
from sklearn.neighbors import KNeighborsClassifier

# X = [study_hours, quiz_score]
X = [
    [1, 40],
    [2, 45],
    [2, 50],
    [4, 75],
    [5, 80],
    [6, 90]
]

# y = 0 means beginner group, 1 means advanced group
y = [0, 0, 0, 1, 1, 1]

model = KNeighborsClassifier(n_neighbors=3)
model.fit(X, y)

new_student = [[3, 60]]
prediction = model.predict(new_student)

print("Prediction:", prediction[0])
```

## Expected Output

```python
Prediction: 0
```

## What It Means

`0` 表示模型预测这位学生更接近 beginner group。

## Tiny App Version

```python
def predict_group(study_hours, quiz_score):
    result = model.predict([[study_hours, quiz_score]])[0]
    return "advanced" if result == 1 else "beginner"

print(predict_group(3, 60))
```

## Try One Change

把这行改成：

```python
print(predict_group(6, 88))
```

看看结果会不会变成 `advanced`。
