## 直接运行

这个例子会预测一位新学生更像 `beginner` 还是 `advanced`。

<CodePlayground
  title="KNN Prediction Demo"
  description="这段代码会找到距离最近的 3 个样本，然后投票得到最终分类结果。你可以直接改 newStudent 再点 Run Code。"
  code={`const trainingData = [
  { input: [1, 40], label: "beginner" },
  { input: [2, 45], label: "beginner" },
  { input: [2, 50], label: "beginner" },
  { input: [4, 75], label: "advanced" },
  { input: [5, 80], label: "advanced" },
  { input: [6, 90], label: "advanced" }
];

function distance(pointA, pointB) {
  const squared = pointA.reduce((sum, value, index) => {
    return sum + (value - pointB[index]) ** 2;
  }, 0);

  return Math.sqrt(squared);
}

function knnPredict(data, newPoint, k) {
  const neighbors = data
    .map((item) => ({
      label: item.label,
      distance: distance(item.input, newPoint)
    }))
    .sort((a, b) => a.distance - b.distance)
    .slice(0, k);

  const votes = {};

  for (const neighbor of neighbors) {
    votes[neighbor.label] = (votes[neighbor.label] || 0) + 1;
  }

  return Object.entries(votes).sort((a, b) => b[1] - a[1])[0][0];
}

const newStudent = [3, 60];
const prediction = knnPredict(trainingData, newStudent, 3);

console.log("New student:", newStudent);
console.log("Prediction:", prediction);`}
/>

## 你可以马上改

- 把 `const newStudent = [3, 60];` 改成 `const newStudent = [6, 88];`
- 再点一次 `Run Code`
- 看结果会不会更接近 `advanced`
