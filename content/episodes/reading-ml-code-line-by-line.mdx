## Algorithm

`LogisticRegression`

## Run This Code

```python
from sklearn.linear_model import LogisticRegression

# X = [watched_minutes, clicked_lessons]
X = [
    [5, 1],
    [8, 1],
    [12, 2],
    [20, 3],
    [30, 4],
    [45, 5]
]

# y = 0 means not buy, 1 means buy
y = [0, 0, 0, 1, 1, 1]

model = LogisticRegression()
model.fit(X, y)

new_user = [[25, 3]]
prediction = model.predict(new_user)
probability = model.predict_proba(new_user)

print("Prediction:", prediction[0])
print("Buy probability:", round(probability[0][1], 2))
```

## Expected Output

```python
Prediction: 1
Buy probability: 0.xx
```

## What It Means

`Prediction: 1` 表示模型预测这位用户更可能购买课程。  
`Buy probability` 是模型给出的购买概率。
